## Project Context

**Project Name:** Furrlytics / 喵谱
**Purpose:** A native iOS app for scientific cat care — tracking feeding, play, training milestones, away time, outings, treats, and deworming schedules. Integrates with Petlibro smart feeders via reverse-engineered API. Supports multiple cats.
**Language**: Swift, SwiftUI
**Data Layer**: SwiftData (local-only for MVP, no backend)
**Minimum iOS Target**: 17.0
**External Integrations**: Petlibro cloud API (unofficial, reverse-engineered)
**Key Services:** PetlibroService, LocationService, NotificationService, OCRService, DateService
**Key Modules:** CatProfile, CareEvent, HomeLocation, DewormingSchedule, TreatItem
